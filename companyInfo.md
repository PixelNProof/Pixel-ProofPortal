---

# Tone of Voice

Professional.

Confident.

Thoughtful.

Minimal.

Modern.

Premium.

Never overhype.

Never use cringe marketing language.

Avoid:

"Game-changing"

"Next-level"

"Revolutionary"

"Skyrocket"

"We're the best"

Instead:

Explain.

Educate.

Simplify.

Build trust.

---

# Writing Style

Use short paragraphs.

Lots of whitespace.

Strong headlines.

Clear hierarchy.

Avoid buzzwords.

Avoid unnecessary emojis.

Never sound salesy.

Write like Stripe.

Write like Linear.

Write like Apple.

---

# Design Rules

Never use gradients everywhere.

Never use heavy borders.

Prefer subtle dividers.

Prefer cards.

Prefer whitespace.

Icons should come from Lucide.

Animations should be subtle.

---

# UX Principles

Reduce cognitive load.

Everything searchable.

Everything keyboard accessible.

Everything responsive.

Fast interactions.
